# 🚗 Vehicle Market Analysis & Automated Audit Pipeline

## 📋 Project Overview
This project performs an exploratory data analysis (EDA) on vehicle pricing data. I utilized **Python**, **Pandas**, and **Seaborn** to identify trends in vehicle depreciation and price distributions.

### 🤖 Special Feature: Automated Governance
Unlike standard portfolios, this project includes a **CI/CD Audit Pipeline**. Every time code is pushed, a custom-built AI Agent (powered by **Gemini 1.5-Flash**) performs a peer-review of the statistical methodology and visualization clarity.

---

## 🛠️ Tech Stack
- **Languages:** Python 3.10
- **Libraries:** Pandas, Seaborn, Matplotlib, Google GenAI SDK
- **Automation:** GitHub Actions (CI/CD)
- **AI Model:** Gemini 1.5-Flash
