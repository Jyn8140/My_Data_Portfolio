## 🚗 Project: Vehicle Market Analysis
**Main File:** 'VEHICLE ANALYSIS PROJECT.ipynb'

### 🎯 Strategic Overview
This project leverages Python to perform **Deep Explaratory Analysis** on vehicle market trends. The goal is to uncover hidden corellations between specific and market valuation.

### 🛠️ Technical Stack
* **Analytics Engine:** Python (Pandas, Numpy)
* **Visualization:** Seaborn, Matplotlib
* **Methodology:** Statistical Analysis and Feature Engineering
### 📈 Project Milestones
1. **Data Wrangling:** Implementing clean-Room techniques to handle null values and outliers.
2. **Multivariate Analysis:** Exploring the no-linear relationship between engine relationship and displacement and efficiency.
3. **Synthesis:** Presenting data-driven conclusions for market stake holders.
