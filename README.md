## 📌 Project Overview
This project performs an exploratory data analysis (EDA) on vehicle market data. As a Data Scientist, my goal was to uncover patterns in vehicle pricing, depreciation, and market trends using Python's robust data ecosystem.

## 🛠️ Technical Stack
* **Language:** Python 3.10
* **Analysis:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Automation:** GitHub Actions (CI/CD)
* **AI Integration:** Gemini 2.0/3 Flash API (Automated Quality Audit)

---

## 🤖 AI-Powered Data Auditing
To ensure high standards of analytical rigor, I built a custom **CI/CD pipeline** that acts as an automated "Senior Data Scientist" peer reviewer.

### How the Automation Works:
1.  **Trigger:** Every code push triggers a GitHub Action.
2.  **Audit:** A Python script (`auto_audit.py`) analyzes the logic within my Jupyter Notebooks via **Gemini 2.0 Flash**.
3.  **Validation:** The AI scans for logic flaws, statistical outliers, or missing data-cleaning steps.
4.  **Reporting:** An audit report is generated automatically to ensure the analysis remains "production-ready."
